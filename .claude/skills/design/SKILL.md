# /design — peça gráfica sob demanda

O "faz-tudo" de design pra peças pontuais. Quando não houver uma skill mais
específica (`/carrossel`, `/thumbnail`, `/capa`, `/banner-design`), use esta.

## O que fazer

1. Definir: tipo de peça, formato/proporção, objetivo e onde vai ser usada.
   Ler `identidade/design-guide.md` (cores, fontes, estilo).
2. Aplicar os fundamentos:
   - **Hierarquia** — uma mensagem dominante, leitura guiada.
   - **Contraste e legibilidade** — funciona no tamanho real de uso.
   - **Espaçamento** — respiro, alinhamento, grid.
   - **Marca** — paleta, fontes e elemento visual coerentes.
3. Gerar a arte no formato pedido. Imagem por IA via `/foto-ia` quando útil.

## O que devolver

1-2 variações + a arte final exportada em `marketing/design/`. Se a peça for
virar padrão recorrente, oferecer transformar num template em `templates/`.

# marketing

O conteúdo que o BuilderIA produz pra sua vitrine pública mora aqui: posts,
carrosséis, artigos, calendário, criativos de anúncio e a biblioteca de prompts.

As skills criam as subpastas sozinhas (ex.: `conteudo/`, `prompts/`,
`ads-criativos/`, `calendario/`). Você não precisa criar nada à mão.
